OpenProcurement complaints buildout
===================================

This software sync tender complaints with mysql backend


### Requirements

- python 2.7
- MySQLdb


### Installation

    $ cd openprocurement.complaints.buildout
    $ python bootstrap.py
    $ bin/buildout


### Configuration

- etc/complaints_queue.ini


### Start / Stop

    $ bin/complaints_init.sh start|stop|restart
